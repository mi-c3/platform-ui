Slider

#Notes

Using Slider from @material/slider

#Props

NameTypeDefaultDescription
labelstringnulllabel
TypographyPropsobject{}props for label wrapper Typography
classNamestringcss class name
namestringname field
valuestringvalue of the field
minnumber0min value in slider bar.
maxnumber100max value in slider bar.
priorityenum: 1,2,3,4,5if priority is set, slider can be on of 5 priority colors
stepnumber1step of changes
disabledboolfalsemake field disabled
discreteboolfalsediscrete slider
showMarkersboolfalseslider with markers
onChangefuncYou can pull out the new value by accessing event.target.value

#How to use

import { Slider } from '@mic3/platform-ui';
<Slider label="Progress" priority={3} value={75} onChange={onChange}  />
