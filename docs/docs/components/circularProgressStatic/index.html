CircularProgressStatic

#Notes

Component CircularProgressStatic for static circular progress bar.

#Props

NameTypeDefaultDescription
sizenumber38circle radius in px
borderWidthnumber3width of border
valuenumber0value can be from 0 to 100
fillColorHEX or CSS colorwhitecolor of bar
foreignObjectContentnodenullrender component inside svg
foreignObjectPropsobject{}properties for svg foreignObject tag, available if foreignObjectContent is not null

#How to use

import { CircularProgressStatic } from '@mic3/platform-ui';
<CircularProgressStatic fillColor="#FFFFFF" value={75} />
<CircularProgressStatic
    fillColor="primary"
    size={40}
    foreignObjectContent={<Avatar {...avatarProps} />}
/>
